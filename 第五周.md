## git 常见需求与总结

- Git 在工作区不能正常显示中文

```bash
$ git status
"\347\254\254\345\233\233\345\221\250.md" # 文件名的中文不能正确地显示，而是显示为八进制的字符编码
# Git配置变量 core.quotepath 设置为 false, 解决不能显示中文的问题
$ git config --global core.quotepath false
```
